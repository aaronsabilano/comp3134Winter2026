1. What is the definition of server port?
A server port is a numbered communication endpoint on a server that is used to identify specific services or applications when data is sent or received over a network.

2. What is server port range?
The server port range goes from 0 to 65535.

3. How many bits does number above represent?
The port number range represents 16 bits.
